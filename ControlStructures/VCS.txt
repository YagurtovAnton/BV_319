Git/GitHub
Git != GitHub

Repository - это хранилище исходных кодов.
fetch - скачивает данные с удаленного репозитория в локальный;
pull  - скачивает и распаковывает данные с удаленного репозитория в локальный;
push  - отправляет данные с локального репозитория на удаленный;

Commit - это контрольная точка, в которую сохраняется текущее состояние репозитория.
COMMIT MESSAGE

Branch - это последовательность commit-ов
master - это главная ветка, на которой происходит mainstream разработки.
merge - слияние веток
